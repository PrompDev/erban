OpenClaw for Business installs itself in one click, parks a little chat box in the corner of your screen, and quietly reads your CRM and drafts your paperwork — so you can check it and send it. No terminal, no config, no IT bloke.

Download for Windows .exe or paste one line:
powershell -NoProfile -Command "irm https://erban.xyz/install.ps1 | iex"

Paste it into Command Prompt, PowerShell, or the Run box (Win+R) — all three work.

Heads up: the .exe is unsigned for now, so Windows SmartScreen shows an "unknown publisher" warning the first time — click More info → Run anyway. Code-signing is on the way.

why it's better

Most "AI tools" hand you a terminal and a config file. This one does the awkward parts itself.

⌘

No terminal, no config

The installer is the setup. It adapts to your exact machine instead of asking you to. If something's off, it fixes it and tells you what it did.

▢

Everything in one folder

It all lands in C:\OpenClawBusiness — app, settings, logs, the lot. Nothing buried in deep system folders. Yours to see, back up, or remove.

✎

Read & draft only

This build reads your CRM and writes drafts for you to check. It cannot send, post, delete or pay — by design, not by promise.
